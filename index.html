Install codecs, software, and more...

Fedy lets you install multimedia codecs and additional software that Fedora doesn't want to ship, like mp3 support, Adobe Flash, Oracle Java etc., and much more with just a few clicks.

To install Fedy in Fedora, open a Terminal and run the following command,

bash -c 'su -c "curl https://satya164.github.io/fedy/fedy-installer -o fedy-installer && chmod +x fedy-installer && ./fedy-installer"'

Everything in an easy to use interface

Fedy has a no-nonsense straightforward UI to get you started as quickly as possible. All the actions are queued so that you don't have to wait for something to finish.
